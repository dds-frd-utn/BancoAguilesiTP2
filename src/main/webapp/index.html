Bienvenidos a Banco Aguilesi

  Para acceder a la informacion de un cliente ingrese a:

http://localhost:8080/Aguilesi/rest/cliente/{IdCliente}

 Para acceder las cuentas ascociadas a un cliente:

http://localhost:8080/Aguilesi/rest/cuenta/deCliente/{IdCliente}

Para acceder a los movimientos de una cuenta ingrese a:

http://localhost:8080/Aguilesi/rest/movimiento/deCuenta/{IdCliente}

Para crear un nuevo movimiento , hacer una peticion POST en la direccion

http://localhost:8080/Aguilesi/rest/movimiento

o para actualizarlo hacer una peticion PUT en esta otra 

http://localhost:8080/Aguilesi/rest/movimiento/{idMovimiento}

con la siguiente estructura:

{"creado": Date

"descripcion": String

"estado": Int                           (1=Pendiente; 2=En Proceso, 3=Procesado, 4=Rechazado)

"idMovimiento": Int              (No necesario al momento de crear un movimiento)

"idCuenta": Int                       (N° de Cuenta)

"importe": Double

tipo":Int }                                 (1=Extraccion, 2=Deposito)

 

 Para obtener el saldo de una cuenta ingrese a: 

http://localhost:8080/Aguilesi/rest/cuenta/saldo/{idCuenta}

Para obtener los movimientos de una cuenta en un estado especifico:

http://localhost:8080/Aguilesi/rest/movimiento/{IdCuenta}/estado/{IdEstado}

Para obtener los movimientos de una cuenta segun una descripcion dada:

http://localhost:8080/Aguilesi/rest/movimiento/{IdCuenta}/{Descripcion}
